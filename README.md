# Black-Jack

The code runs 1000 games 100 times, from this it calculates the points (based of if the game was won / lost / drawn. The first run has
a random chance of hitting ( getting an extra card). This method achieves  an average point score (per 1000 games) of -268.
![Optional Text](../master/Images/RandomChanceAttempt1.PNG)
Based of the previous results the code runs the same number of games but this time reference the previous results to see if this hand
has happened before and if so what was the results if it hit / didn't hit; from here it can decide whats best. This method achieves 
an average point score (per 1000 games) of -11.
